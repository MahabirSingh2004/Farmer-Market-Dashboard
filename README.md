# Farmer Market Dashboard (Power BI)

## Project Overview
This project is an interactive **Power BI dashboard** developed using **Power BI Desktop** to analyze **Farmer Market data**.  
The dashboard provides insights into **sales, quantity, and their relationship**, helping users understand market performance through visual analytics.

---

## Objectives
- Analyze farmer market data using different perspectives  
- Understand sales and quantity trends  
- Compare product and market performance  
- Support data-driven decision-making using interactive dashboards
